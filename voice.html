Voice Platform

With the help of the voice platform, I have created a voice-based platform which can be used to convert the voice into text and can also detect the sentiment of the voice. The tool was deployed by using the Flask API with dockers containers and kubernetes. Pre-trained models were used to detect the sentiment of the voice.

  • Developed Microservices for web request handling and S3 bucket management.
  • Handle S3 bucket and AWS transcribe services.
  • Implement pre-trained BERT model in the current model for better accuracy.
  • Highlight the sentiments(mood) of the user, by it statements.
  • Leveraging Databricks service, using of Dockers & Kubernetes for deployment.
